## STAR Method Reflections

### 1. Reflection on coding in Markdown
**Situation:** Display my CV on a GitHub pages website


**Task:** I was asked to create a markdown file that would display the contents of my CV in a readable format for viewing on my github website. The actual task was easier than expected, with a guide already being provided on the task sheet. 

**Action:** All of the information that was used came directly from my CV. The part that took the most time was validating that the layout of the website viewable CV looked correctly when displayed on the website. I use VS Code as an IDE, and it has a built-in markdown viewer which made checking changes immediate.

**Result:** The CV is now live on the website, and displays exactly how I expected it to.

**Situation:** Demonstrate my interviewing skills
